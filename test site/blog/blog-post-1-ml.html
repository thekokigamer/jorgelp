My First Dive into Machine Learning

Posted on October 26, 2023 by Your Name

This week, I started exploring the fascinating world of machine learning. My initial focus has been on understanding supervised learning algorithms like linear regression and decision trees. The math can be challenging, but seeing the practical applications is incredibly motivating.

I began by working through Andrew Ng's classic Machine Learning course on Coursera, which provides a solid theoretical foundation. To get hands-on, I've been playing with Scikit-learn in Python, a fantastic library that makes implementing these algorithms surprisingly straightforward.

Key Concepts I've Grasped So Far:

  • Supervised Learning: Training a model on labeled data (input-output pairs).
  • Linear Regression: Finding the best-fit line to predict continuous values. I practiced this by predicting housing prices based on features like size and number of bedrooms from a sample dataset.
  • Decision Trees: Tree-like models of decisions and their possible consequences, used for both classification and regression. Visualizing these trees helped me understand how they make predictions.
  • Model Evaluation: Understanding metrics like Mean Squared Error (MSE) for regression and accuracy/precision/recall for classification.

One of the "aha!" moments was realizing how feature scaling (like normalization or standardization) can significantly impact the performance of certain algorithms, especially those that rely on distance calculations.

Challenges and Next Steps:

The biggest challenge right now is internalizing the mathematical underpinnings, especially for more complex algorithms. My next steps are to delve into:

  • Logistic Regression for binary classification problems.
  • Understanding overfitting and techniques to combat it, like cross-validation and regularization.
  • Exploring a more complex dataset and trying to build a predictive model from scratch.

It's a steep learning curve, but I'm excited to see where this journey into ML takes me and how I can apply these skills to engineering problems in the future!
